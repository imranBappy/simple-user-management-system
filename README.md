# User Management System

Live Link [https://user-m-s.netlify.app](https://user-m-s.netlify.app)

## Getting Started

First, run the development server:

```bash

cd server
npm install
# then
npm run dev

# and

cd client
npm install
# then
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Rest API

```
Root Url: https://simple-user-management-system-u42w.onrender.com

GET /users
GET /users/:id
POST /users
PUT /users/:id
DELETE /users/:id

```
